# Write the Weekly Email

**The job:** Draft the Monday email to the list (~1,400 designers and small-brand founders) — one value-forward message, sent every Monday morning.

## Boot chain (read these, in order, and you have the skill)
1. This note, end to end.
2. [[Brand Voice Guide]] — how the studio sounds: warm, plain, one idea at a time.
3. [[Active Priorities]] — whether there's a launch, opening, or promo to feature this week.
4. [[Client Roster]] — only if the email references live client work; anonymize everything.

## The procedure
1. Check [[Active Priorities]] for anything to feature (course milestones, open client slots). If nothing is live, pick an evergreen craft angle from recent studio work.
2. Pull the tone from the [[Brand Voice Guide]]; keep the hook different from the last few sends.
3. Draft it: one idea, one story or tip, one clear call to action. Plain text, around 200 words.
4. Write the subject line and a one-line preview.
5. Save the draft to this folder as `Email Draft YYYY-MM-DD.md` and tell Maya it's ready to review. Never schedule or send it — Maya sends.

## Quality bar
- One message, one idea. No kitchen-sink emails.
- It sounds like the Voice Guide, not like an AI. No em-dashes.
- A specific subject line, never a vague one.
- Never invent an offer or a date — use only real ones from [[Active Priorities]].
- Client work is anonymized unless Maya has written approval to name them.

## Lessons (fold corrections in here over time)
- Get to the value in the first sentence; cut the throat-clearing intro.
- Always end with a P.S. that repeats the call to action.
- Course mentions perform better as one casual line than as a hard pitch (learned 2026-06-15).
