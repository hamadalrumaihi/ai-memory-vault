# Investigative Method

How I work a case. The thread I start with decides the method — not the other way around. Any AI assisting on a case reads this first and suggests the next move based on which thread we're holding.

## The four threads

**1. Plate up.** Start from a license plate and work upward: registered owner, known associates, prior stops, where the vehicle shows up on [[MDT]] history.

**2. Name out.** Start from a name and go looking: priors, addresses, employers, who they run with. Canvass — go around and find them.

**3. Appearance across priors.** Match a description against previous subjects — same attire, same build, same MO. Crews keep their look; that's a thread.

**4. Area and pattern.** Start from where it happened: what crimes are common in that area, which crews work it, what the pattern says about who and when.

## Working rules

- Log every thread pulled in the case note, even dead ends — a dead end today is a cross-reference tomorrow.
- Facts in the official narrative; theories and hunches in the case file's working notes, clearly marked.
- At every case review, the assistant lists what hasn't been tried yet from the four threads above.
